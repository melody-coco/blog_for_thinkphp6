



### 搭建Hexo博客 [hexo官网](https://hexo.io/zh-cn/docs/)

#### 准备工作

+ 下载`nodejs`环境

+  配置`npm`

  - 安装 `cnpm` 

    `npm install -g cnpm --registry=https://registry.npm.taobao.org`

+ 安装 `hexo-cli`
	
	- `cnpm install hexo-cli -g`
+ 在github里面创建一个仓库,  仓库名字是 `用户名.github.io`(下面的README.md初始化勾选)
